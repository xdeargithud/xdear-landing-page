## 🛠️ Tech Stack

- **HTML5** — struktur halaman
- **CSS3** — styling, animasi, dan responsive layout
- **Vanilla JavaScript** — interaksi dan animasi dinamis
- **Lucide Icons** — library icon open source
- **Simple Icons** — library icon brand/sosial media
- **Vercel** — hosting dan deployment

---

## 📄 Fitur

- **Hero Section** — headline, typing animation, statistik bisnis, dan mockup foto dengan aura gradient berputar
- **Marquee Section** — slider foto OWN dan PT dari xdear store
- **About Section** — profil bisnis, banner video, foto profil, expand cards galeri, dan about cards
- **Why Us Section** — 6 alasan memilih xdear store dalam format card grid
- **Service Section** — 6 layanan dalam format card slider dengan mesh gradient
- **Pricing Section** — 5 paket harga dalam format card slider dengan tombol CTA WhatsApp
- **Cara Kerja MC Section** — stepper horizontal 6 langkah dengan foto asset dan tabel fee admin
- **Review Section** — testimonial horizontal auto-scroll
- **Expand Gallery Section** — 10 foto expandable dengan lightbox preview
- **FAQ Section** — accordion 6 pertanyaan umum
- **CTA Section** — kontak WhatsApp dan peringatan anti-clone
- **Footer** — navigasi, sosial media, metode pembayaran, dan copyright

## ✨ Highlights

- Dark mode modern dengan tema warna ungu, cyan, dan pink
